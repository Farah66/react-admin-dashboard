# 📊 React Admin Dashboard

A fully responsive **Admin Dashboard Application** built with **React.js** and **Material UI**, featuring modern analytics, charts, tables, calendar management, and light/dark theme support.

This project is based on the popular dashboard tutorial by **Ed Roh** and demonstrates how to build production-level admin interfaces with reusable React components. :contentReference[oaicite:0]{index=0}

---

## 🚀 Features

✅ Fully Responsive Dashboard Layout  
✅ Light & Dark Mode Theme  
✅ Sidebar Navigation Menu  
✅ Analytics Overview Cards  
✅ Interactive Charts & Graphs  
✅ User Management Tables  
✅ Calendar Scheduling System  
✅ FAQ Section  
✅ Form Validation  
✅ Clean UI / UX Design
